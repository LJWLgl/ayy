### 项目介绍
ayy(爱易园)基于微信小程序开发的，服务校园二手交易，目前包含小程序端和后端，前端是基于小程序原生开发，后端使用SpringBoot搭建，主要包含以下功能：
+ 商品信息发布与浏览
+ 社交（话题、评论）
+ 商品搜索
+ 位置定位
+ 图片上传
+ 手机号绑定
+ 计数系统
+ ...
